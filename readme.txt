Awesome plugin to display slideshow on your wordpress site.

== Description ==
The **Ultimate Slideshow plugin** allows easy and quick way to set up a slideshow on any page.

Simple and flexible, Ultimate Slideshow works with any WordPress theme. 
Easy to setup. 

	
 = Free VERSION (Core) Features =
    1. Admin can create various slideshows and upload images for displaying in the slideshow. 
    2. Admin/Author/Editor can click on "myslideshow" button displayed on the editor to enter the shortcode.
	
 = Display Features =
    1. Page which has shortcode "[myslideshow id='']" displays the images in the slider.
    
== Installation ==

Manual Installation

1. Upload the `ultimate-slideshow` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Visit Settings tab of the plugin and enter "ultimate slideshow".

4. After you have setup your images you can sort them in the order you want to display in frontend.

5. After you have added images, go to "Pages" inside your admin dashboard and add a new page.

6. Enter this text "[myslideshow]" as a shortcode inside this new page and publish it. 
This page is responsible for displaying slideshow of images set via admin.
